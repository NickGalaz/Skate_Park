GUIA DE USO PARA ADMIN
#######################
Tras levantar el servidor con node index.js:

Paso 1
Ingresar a la página http://localhost:3000/admin
-------------------------------------------------

Paso 2
Al visualizar la lista de usuarios registrados, en la columna de estado se debe clickear el checkbox, para autorizar o desautorizar usuarios.
Un tik en el checkbox marca que el usuario está autorizado y si el checkbox está vacío, es porque el usuario no está autorizado.
